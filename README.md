# Computer Scientist Foundation

A personal repository for Computer Science foundations and SDE practices, inspired by **NAB Wecamp Sessions**.

## Goal
Build a solid foundation for an **SDE** (Software Development Engineer) role:
- **OS & Scripting:** Linux, Bash, Python automation.
- **Networking:** Protocols, SSH, and Security.
- **Infrastructure:** IaC (Terraform, Ansible).
- **Automation:** CI/CD pipelines.
- **Cloud:** Containers (Docker/K8s) & AWS/Azure.

## Roadmap
- [ ] **Phase 1:** Linux, Shell Scripting & Python.
- [ ] **Phase 2:** Networking & Security Basics.
- [ ] **Phase 3:** CI/CD (GitHub Actions, Jenkins).
- [ ] **Phase 4:** Infrastructure as Code (Terraform).
- [ ] **Phase 5:** Docker & Kubernetes.
- [ ] **Phase 6:** Cloud (AWS/Azure) & Monitoring.
