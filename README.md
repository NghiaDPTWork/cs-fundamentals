# Computer Scientist Foundation

Chào mừng bạn đến với repo học tập cá nhân về **Computer Science Foundations**. Đây là nơi lưu trữ lộ trình, ghi chú và mã nguồn thực hành tập trung vào các nguyên lý cốt lõi của khoa học máy tính, được ứng dụng thực tế thông qua lộ trình học **NAB Wecamp Sessions**.

## Mục tiêu

Repo này tập trung vào việc xây dựng nền tảng vững chắc để trở thành một SDE, bao gồm:

- **Hệ điều hành & Scripting:** Làm chủ Linux và tự động hóa với Bash/Python.
- **Networking & Security:** Hiểu rõ các giao thức mạng và bảo mật hệ thống.
- **Infrastructure as Code (IaC):** Quản lý hạ tầng bằng mã (Terraform, Ansible).
- **CI/CD & Automation:** Xây dựng luồng tích hợp và triển khai tự động.
- **Containerization:** Đóng gói ứng dụng với Docker và Kubernetes.

## Lộ trình thực hiện

Dự án được triển khai theo các giai đoạn cốt lõi của DevOps:

- [ ] **Giai đoạn 1:** Linux Fundamentals, Shell Scripting & Python.
- [ ] **Giai đoạn 2:** Networking (HTTP, DNS, SSH) & Security Basics.
- [ ] **Giai đoạn 3:** CI/CD Pipelines (GitHub Actions, Jenkins).
- [ ] **Giai đoạn 4:** Infrastructure as Code (Terraform, CloudFormation).
- [ ] **Giai đoạn 5:** Containers (Docker) & Orchestration (Kubernetes).
- [ ] **Giai đoạn 6:** Monitoring, Logging & Cloud (AWS/Azure).
